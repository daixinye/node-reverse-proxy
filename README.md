# Node-Reverse-Proxy

A reverse proxy with node.js

## Usage

### start proxy
```
npm run start
```

## Config Commands
### get configs
get all configs
```
npm run getall
```

get specific config
```
npm run get HOSTNAME
```

### set config
```
npm run set HOSTNAME PORT
```

### delete config
```
npm run del HOSTNAME
```

## Default

when the host is IP, you can set the DEFAULT port as the server.

### set default
```
npm run setDefault PORT
```
