# Node-Reverse-Proxy

Simple http reverse proxy cli tool

## Install

```shell
$ npm install nrp-cli -g
```

## Usage

### start proxy
need sudo
```shell
$ sudo nrp-cli start
```

## Config Commands
### list configs
list all configs
```shell
$ nrp-cli list
```

### set config
```shell
$ nrp-cli set <hostname> <port>
```

### delete config
```shell
$ nrp-cli del <hostname> <port>
```
